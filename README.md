# HotelProject-TechGii

Tri Gong Hotel is a small hotel showcase and booking prototype built with React, TypeScript, and Vite.

## Project Structure

- `frontend/` contains the hotel landing page and booking UI
- `backend/` contains a minimal Express API scaffold for future booking endpoints

## Frontend

Run the frontend locally:

```bash
cd frontend
npm install
npm run dev
```

Build and lint:

```bash
npm run build
npm run lint
```

## Backend

Run the backend locally:

```bash
cd backend
npm install
npm run dev
```

The backend currently exposes:

- `GET /api/health`

## Notes

- Generated folders like `node_modules/` and `dist/` are ignored
- Hotel images used by the UI live in `frontend/src/assets/`
